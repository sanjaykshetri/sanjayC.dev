# sanjayC.dev

Professional portfolio site for Sanjay Chhetri.

## Focus

Data Scientist | Machine Learning | Behavioral AI

This site highlights applied projects in trust analytics, NLP, quantitative research, and product-facing machine learning.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Run lint checks:

```bash
npm run lint
```

3. Build production bundle:

```bash
npm run build
```

4. Start local dev server:

```bash
npm run dev
```

Visit http://localhost:3000

## Core Routes

- `/` Home
- `/portfolio` Projects and featured research
- `/blog` Writing
- `/about` About
- `/resume` Resume download and profile links
- `/contact` Contact

## Deployment

Deploy on Vercel using default Next.js settings.
